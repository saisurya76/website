Install & Run

Key API Endpoints

POST /auth/signup

Returns access_token to use as Bearer.

POST /api/contexts

Multipart upload of one/more files ‚Üí creates a context.

GET /wrappers/api/contexts/{id}/wrapper

Returns a real PNG wrapper with QR + embedded locator.

POST /wrappers/api/contexts/resolve-from-wrapper

Upload wrapper PNG or JSON locator ‚Üí returns context_id and enc_url.
